# Public Leaderboard Starter Split

This split is a small public benchmark intended for initial curation and local
experimentation.

Targets:

- `7ftv_A` - 351 aa
- `8cny_A` - 182 aa
- `8g8r_A` - 426 aa
- `8i85_A` - 280 aa

Suggested usage:

- keep `8cny_A` and `8i85_A` in the tight-loop dev tier
- use `7ftv_A` as a medium-length public target
- use `8g8r_A` as a stretch / stress target

For production, the public leaderboard and private final sets should be expanded
beyond these four bundled examples.
