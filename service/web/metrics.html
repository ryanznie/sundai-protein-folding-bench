Metrics Glossary

Definitions and expected ranges for structure quality metrics.

Metric Range Description
TM-score 0.0 → 1.0 Template Modeling score. Higher is better. Scores > 0.5 generally indicate a correct global fold.
lDDT 0.0 → 1.0 Local Distance Difference Test. Measures local structure consistency. Scores > 0.7 indicate high confidence.
CA-RMSD Å (Angstroms) Root Mean Square Deviation of C-alpha atoms. Lower is better. < 2.0Å is high accuracy.
Runtime Seconds Total time taken for inference across all targets in the benchmark bundle.
